## 1. ASPサイト審査と提携までの流れ

### ASP登録と広告主審査は別

| ステップ | 内容 | 審査の有無 |
|----------|------|-----------|
| ASP登録 | A8.netなどに会員登録する | A8.netは審査なし（誰でも登録可） |
| サイト登録 | 運営するブログ/サイトをASPに登録する | サイトの基本要件チェックあり |
| 広告主への提携申請 | 掲載したい案件に申請する | **広告主ごとに審査あり**（ここが本番） |

### 広告主審査で見られるポイント

1. **コンテンツの質と量**: 最低10記事程度、各1,000文字以上の独自コンテンツ
2. **テーマの一致**: サイトのテーマと広告の商材・ターゲットが合っているか
3. **更新頻度**: 長期間更新されていないサイトは審査落ちしやすい
4. **法令遵守**: 薬機法・景表法に違反する表現がないか
5. **必須ページの設置**: プライバシーポリシー、免責事項、運営者情報、お問い合わせ

### 審査落ちでよくある理由

- 開設から日が浅い、コンテンツが少ない
- 古い情報や正しくない情報が掲載されている
- プライバシーポリシー・運営者情報がない
- 広告主のテーマとサイトの内容が合っていない

## 2. ステルスマーケティング規制（PR表記義務）

### 概要

2023年10月1日施行。景品表示法の「不当表示」にステルスマーケティングが追加された。**広告であることを隠して消費者に商品を推薦する行為は違法**。

### アフィリエイトサイトへの影響

アフィリエイトリンクを含む記事はすべて「広告」に該当するため、**PR表記が必須**。

### 表記方法

**使用可能な文言:**
- 「広告」「PR」「プロモーション」
- 「アフィリエイト広告を利用しています」
- 「本ページはプロモーションが含まれています」
- 「PR（提携リンクを含みます）」

**表記位置のルール:**
- ページのファーストビュー（スクロールなしで見える位置）に配置
- 本文中のアフィリエイトボックスにも個別に明示
- 文字サイズや色が周囲に紛れて読めない状態はNG

### 現在の実装状況

本サイトでは `AffiliateBox` コンポーネントに「PR（提携リンクを含みます）」の表記を実装済み。ただし、**ページ上部（ファーストビュー）にも表記が必要**な点は追加対応が望ましい。

### 違反時のリスク

- 措置命令（消費者庁）
- 措置命令違反時: **2年以下の懲役または300万円以下の罰金**

> 参考: [消費者庁 ステルスマーケティング規制](https://www.caa.go.jp/policies/policy/representation/fair_labeling/stealth_marketing/)

## 3. 景品表示法（景表法）

### 概要

商品・サービスの不当な表示を規制する法律。アフィリエイターが作成した広告も**広告主の表示とみなされる**ため、違反すれば広告主にも影響が及ぶ。

### 優良誤認表示（品質に関する嘘）

商品やサービスの品質・効果を事実より著しく優良であると誤認させる表示。

| NG例 | 理由 |
|------|------|
| 「このクリニックは100%治る」 | 医学的にあり得ない保証表現 |
| 「業界No.1の効果」（根拠なし） | 客観的根拠がない最上級表現 |
| 「他社より圧倒的に効果が高い」 | 比較の根拠が不明確 |

### 有利誤認表示（価格・条件に関する嘘）

取引条件を事実より著しく有利であると誤認させる表示。

| NG例 | 理由 |
|------|------|
| 「今だけ50%オフ！」（常にその価格） | 通常価格を偽った二重価格表示 |
| 「先着100名限定」（実際は制限なし） | 虚偽の限定表示 |
| 「他社比較で最安」（一部条件を隠して比較） | 恣意的な比較 |

### ランキング・比較記事での注意点

- **根拠のないランキング**は違法。「人気ランキング」と称して報酬額順に並べる行為はNG
- 比較は**公正・正確**なデータに基づくこと
- 比較条件を明示すること（何を基準にしたランキングか）
- 特定の商品を不当に有利/不利に扱わないこと

> 参考: [消費者庁 景品表示法](https://www.caa.go.jp/policies/policy/representation/fair_labeling/)

## 4. 薬機法 — スキンケア・化粧品の広告ルール

### 概要

薬機法（医薬品、医療機器等の品質、有効性及び安全性の確保等に関する法律）は、医薬品・化粧品・健康食品等の広告を規制する。**化粧品は標ぼう可能な効能効果が56項目に限定されている**。

### 化粧品で言えること（主要な効能効果）

以下は厚労省が認めた化粧品の効能効果56項目のうち、スキンケアに関連するもの:

- 肌を整える / 肌のキメを整える
- 肌荒れを防ぐ
- 肌をひきしめる
- 肌にうるおいを与える / 肌の乾燥を防ぐ
- 肌にはりを与える / 肌にツヤを与える
- 肌をなめらかにする
- 肌を清浄にする
- 肌を健やかに保つ
- 日やけを防ぐ / 日やけによるシミ・ソバカスを防ぐ
- **乾燥による小ジワを目立たなくする**（効能評価試験済みの場合のみ）

### 絶対に言ってはいけないこと

| NG表現 | 理由 |
|--------|------|
| 「ニキビが治る」「ニキビを改善」 | 治療効果は医薬品の領域 |
| 「シミを消す」「シミが薄くなる」 | 化粧品の効能範囲外 |
| 「シワをとる」「シワが消えた」 | 化粧品の効能範囲外 |
| 「アンチエイジング」 | 老化防止は医薬品的効能 |
| 「美白になる」（肌の色自体を白くする意味で） | 医薬部外品でも「メラニンの生成を抑え、しみ・そばかすを防ぐ」までが限度 |
| 「浸透する」（肌の奥深くまで） | 化粧品は角質層までしか作用しない |
| 「毛穴を引き締める」 | 「毛穴を目立たなくする」ならOK |
| 「アトピーの方におすすめ」 | 疾患名の表示はNG |
| 「医師推薦」「皮膚科医おすすめ」 | 医療関係者の推薦はNG |
| 「使用前後の写真」（効果を示す目的） | Before/Afterは効能保証とみなされる |

### OK表現の言い換え例

| NG → OK |
|---------|
| 「ニキビに効く」→「肌荒れを防ぐ」 |
| 「シミを消す」→「日やけによるシミを防ぐ」 |
| 「シワがなくなる」→「乾燥による小ジワを目立たなくする」（試験済みの場合） |
| 「美白になれる」→「肌のトーンを明るく見せる」 |
| 「毛穴を引き締める」→「肌をひきしめる」「毛穴を目立たなくする」 |
| 「保湿効果がすごい」→「肌にうるおいを与える」「ベタつきが少ない」 |

### 体験談の書き方

- **効果効能の体験談はNG**: 「たるみが消えた」「保湿効果を実感」
- **使用感の感想はOK**: 「ベタつきが少ない」「肌にスーっとなじむ」「さっぱりした使用感」
- 感想はあくまで**使用感（テクスチャー・香り・使い心地）**に限定する

### 違反時のペナルティ

- 虚偽・誇大広告: **2年以下の懲役または200万円以下の罰金**
- 課徴金: 違反で得た売上の **4.5%**
- 行政処分: 広告差し止め、販売中止、業務停止

> 参考: [A8.net 薬機法ガイド](https://www.a8.net/compliance/pmd.php)、[薬事法ドットコム 化粧品広告](https://www.yakujihou.com/knowledge/cosme-advertisement/)

## 5. 医療広告ガイドライン — AGA治療・医療脱毛の広告ルール

### 概要

厚生労働省が定めた医療広告に関する指針。医療は「人の生命や身体に関わるサービス」であるため、他分野より**極めて厳しく**規制される。

### アフィリエイトが医療広告に該当する条件

以下の**両方**を満たす場合、アフィリエイト記事も医療広告に該当する:

1. **誘引性**: 患者の受診を誘引する意図がある
2. **特定性**: 医療機関の名称が特定可能

→ クリニック名を出してアフィリエイトリンクを貼る行為は、ほぼ確実に医療広告に該当する。

### 重要: バナー広告は「限定解除」の対象外

医療機関のWebサイト自体は一定条件下で情報提供が認められる（限定解除）が、**アフィリエイトのバナー広告やテキスト広告はこの限定解除の対象外**。つまり、医療広告ガイドラインで禁止されている表現は**一切使えない**。

### 禁止される8つのカテゴリ

1. **広告不可事項の広告** — 広告できない内容を広告する
2. **虚偽広告** — 「絶対に安全」「治療成功率100%」
3. **比較優良広告** — 「日本一の病院」「A院◎、B院△」
4. **誇大広告** — 事実を著しく拡大した表現
5. **患者の体験談** — 「痛くなかった」「おすすめ」（主観的感想すべてNG）
6. **Before/After写真** — 治療前後の写真は誤認を招くためNG
7. **公序良俗違反** — 品位を損なう内容
8. **その他** — 費用の過度な強調、期間限定キャンペーン等

### AGA治療記事での具体的なNG/OK

| NG表現 | OK表現 |
|--------|--------|
| 「○○クリニックは痛くなかった」 | 一般的なAGA治療の種類・仕組みの解説 |
| 「××院長の治療は丁寧でした」 | 治療法（フィナステリド等）の一般的な説明 |
| 「この病院はおすすめです」 | 「AGA治療薬を取り扱っております」（医薬品非特定） |
| 「絶対安全な手術です」 | 自由診療である旨と標準的な費用の併記 |
| 「今なら○円でキャンペーン中」 | クリニックの所在地・診療時間の事実情報 |
| 「著名人も推薦しています」 | 治療にかかる一般的な費用相場の説明 |
| 「治療成功率100%」 | 治療の副作用に関する一般的な注意喚起 |
| 「最新マシーン○○で施術」（機器名特定） | 「ジェネリック医薬品を採用」（医薬品非特定） |
| 「医薬品『○○錠』を処方します」（医薬品名特定） | 医療機関の基本情報（住所、電話番号等） |

### 医療脱毛記事での注意点

- クリニック名を出した時点で医療広告の規制を受ける
- 「痛みが少ない」等の体験談はNG
- レーザー機器の具体的な名前を出すことは慎重に（機器名の特定は規制対象の可能性）
- 料金表示は事実に基づき、「お得」「安い」等の有利誤認表現を避ける

### 本サイトの記事構成への影響

本サイトの想定記事構成（体験レビュー + 比較ランキング）は医療広告ガイドラインと**大きく衝突する**。以下の対応が必要:

1. **体験談**: 特定クリニックと紐づけず、「AGA治療全般の体験」として書く
2. **ランキング**: 特定クリニックの優劣比較はNG。事実情報（費用・所在地・診療時間）の一覧に留める
3. **CTA**: 「おすすめです」ではなく「公式サイトで詳しく見る」「無料相談を予約する」に留める
4. **記事構成の分離**: 治療法の一般解説記事（アフィリエイトなし）と、クリニック情報記事（最小限の事実情報 + アフィリエイト）を分ける

> 参考: [A8.net 医療広告ガイド](https://www.a8.net/compliance/medical-ad.php)、[厚生労働省 医療広告ガイドライン（PDF）](https://www.mhlw.go.jp/content/001304521.pdf)

## 6. ASP別ルール（A8.net / もしもアフィリエイト）

### A8.net 主要禁止事項

| カテゴリ | 禁止内容 |
|----------|---------|
| 不正クリック | 自分でクリック、相互クリック、bot使用 |
| 不正申込 | 自己アフィリ（許可案件以外）、虚偽情報での申込、冷やかし |
| 広告素材改変 | バナー画像の変更、テキスト広告の文言変更、リンクの改変 |
| スパム行為 | 掲示板投稿、未承諾メール配信、コメントスパム |
| 著作権侵害 | 他サイトのコンテンツ無断転用、芸能人画像の無断使用 |
| なりすまし | 広告主の公式サイトを名乗る行為 |
| 成果条件の公開 | 報酬額・成果条件のスクリーンショット公開 |
| 複数アカウント | 同一人物/法人の複数アカウント運営 |

**違反時の対応:**
- 不正な成果への報酬不払い
- 過去支払済み報酬の返還請求
- 強制退会（未払い報酬全額没収、二度と登録不可）
- 悪質な場合は法的措置・刑事訴訟

### もしもアフィリエイト 主要ルール

| カテゴリ | 内容 |
|----------|------|
| コンテンツ制作期限 | 登録から6ヶ月以内にコンテンツを制作する必要あり |
| 広告素材改変禁止 | 提供された素材を無許可で改変してはならない |
| 報酬行為の強制禁止 | ユーザーに申込を強制・依頼してはならない |
| 虚偽情報禁止 | プロフィール情報の虚偽記載はNG |
| 著作権遵守 | 自作でない文章・写真の無断使用禁止 |
| 引用ルール | 自身の記述が「主」で引用が「従」、出典明記、改変なし |
| 翻案権 | ドラマ・映画等のストーリー全文書き起こしはNG |
| 肖像権 | 有名人写真の無断使用禁止 |
| ランキング | 報酬額順のランキングを「人気ランキング」と偽るのはNG |

> 参考: [A8.net 禁止事項](https://www.a8.net/compliance/prohibited-matter.php)、[もしもアフィリエイト 利用規約](https://af.moshimo.com/af/www/terms/shop)、[もしもアフィリエイト 不正事例](https://af.moshimo.com/af/www/about/unfaircase)

## 7. サイト運営上の必須対応チェックリスト

### Phase 1: ASP審査前に準備すべきこと

以下はASPに登録申請する**前**の段階で完了させる。

- [x] **プライバシーポリシー** (`/privacy`) — Cookie利用、GA開示、運営者情報を含む
- [x] **免責事項** (`/disclaimer`) — 医療情報免責、リンク先免責を含む
- [x] **お問い合わせ** (`/contact`) — Google Forms埋め込み
- [x] **Cookie通知バナー** — 画面下部に表示
- [ ] **独自コンテンツ** — 最低10記事、各1,000文字以上のアフィリエイトリンクを含まない情報記事
- [ ] **Google Search Console** — サイトマップ送信済み
- [ ] **1ヶ月以上の運営実績** — 定期的な記事更新の履歴

> **注意**: この段階では `NEXT_PUBLIC_AFFILIATE_ACTIVE` は設定しない（PR表記は非表示のまま）。プライバシーポリシー・免責事項の文言は「今後参加する場合があります」の将来形になっている。

### Phase 2: ASP登録 → サイト審査 → 広告主への提携申請

1. A8.net に会員登録（審査なし）→ サイトを登録
2. もしもアフィリエイトに会員登録 → サイトを登録
3. 掲載したい案件（広告主）に提携申請 → **広告主ごとに審査あり**
4. 審査通過を待つ（数日〜数週間）

### Phase 3: ASP承認後にやること（対応チェックリスト）

広告主との提携が承認されたら、以下を**すべて**対応してからアフィリエイト記事を公開する。

#### 環境変数の変更

- [ ] `.env.local`（ローカル）と Vercel の環境変数に `NEXT_PUBLIC_AFFILIATE_ACTIVE=true` を追加
- [ ] 再デプロイして反映を確認（ヘッダー直下にPR表記バー、フッターにアフィリエイト注記が表示される）

#### プライバシーポリシーの文言更新

- [ ] `src/app/privacy/page.tsx` のアフィリエイトセクションを将来形→現在形に変更:
  - 「今後アフィリエイトプログラム…に参加し、…掲載する場合があります」
  - → 「以下のアフィリエイトプログラムに参加しています。…掲載しています」
  - 参加中のASP名を具体的にリストアップ

#### 免責事項の文言更新

- [ ] `src/app/disclaimer/page.tsx` の広告セクションを将来形→現在形に変更:
  - 「今後アフィリエイトプログラムを利用して…紹介する場合があります」
  - → 「アフィリエイトプログラムを利用して商品・サービスを紹介しています」

#### 記事コンテンツの準備

- [ ] microCMSで記事の `affiliateLinks` フィールドにASPから取得したリンクを設定
- [ ] 記事本文中のインラインリンクも実際のASPリンクに差し替え
- [ ] 各リンクの `rel` 属性に `nofollow sponsored` が含まれていることを確認

## 9. 食事・健康法カテゴリの法規制ガイド

本セクションは、「食事・健康法」カテゴリ（身体に良い食べ物・飲み物の効果、グルテンフリー、糖質制限など）のコンテンツを作成する際に遵守すべき法規制をまとめたものである。

> **前提**: 健康食品には法律上の定義がない。「一般食品」として販売されるものに対して、薬機法・健康増進法・景品表示法・食品表示法・特定商取引法が複合的に規制をかける。食品で「効く」「治る」は絶対にNG。

---

### 9.1 健康増進法 — 虚偽誇大広告の禁止（第65条）

#### 概要

健康増進法第65条第1項は、「何人も、食品として販売に供する物に関して広告その他の表示をするときは、健康の保持増進の効果その他内閣府令で定める事項について、著しく事実に相違する表示をし、又は著しく人を誤認させるような表示をしてはならない」と定めている。

#### 重要ポイント

- **「何人も」規制**: 広告主だけでなくアフィリエイターも対象
- **対象**: 錠剤・カプセル形状の食品だけでなく、野菜・果物・調理品等すべての食品
- **判断基準**: 特定の用語・文言の使用を一律に禁止するものではなく、「表示全体」から個別具体的に判断される
- **虚偽誇大表示を行ったアフィリエイターも違反に該当する**

#### 規制される「健康保持増進効果」の範囲

| 効果の種類 | 例 | 規制の有無 |
|-----------|------|----------|
| 疾病の治療・予防 | 「糖尿病の予防に」「がん予防」「血圧を下げる」 | **NG** |
| 身体の組織機能の増強・増進 | 「免疫力アップ」「脂肪燃焼」「代謝促進」 | **NG** |
| 特定の保健の用途 | 「おなかの調子を整える」「コレステロールを下げる」 | **許可なしNG**（トクホ・機能性表示食品のみ可） |
| 栄養成分の含有表示 | 「ビタミンC 100mg配合」「食物繊維が豊富」 | **OK**（事実なら可） |
| 一般的な栄養補助 | 「不足しがちな栄養を補う」「栄養バランスを整える」 | **OK** |

> 参考: [消費者庁 健康増進法（誇大表示の禁止）](https://www.caa.go.jp/policies/policy/representation/extravagant_advertisement)

---

### 9.2 薬機法 — 健康食品が「医薬品」扱いされないために

#### 概要

健康食品は医薬品ではないため薬機法の直接的な規制対象ではないが、**医薬品的な効能効果を標ぼうした時点で「未承認医薬品の広告」として薬機法違反になる**。

#### 医薬品と判断される4つの基準

| 基準 | NG例 | 補足 |
|------|------|------|
| **成分** | 医薬品成分リストに掲載された成分の使用 | 医薬品専用成分を含む食品は未承認医薬品扱い |
| **剤型** | アンプル剤、舌下錠、舌下滴下 | 錠剤・カプセルは食品でも可 |
| **用法・用量** | 「食事の前に3粒」「寝る前に服用」 | 医薬品的な服用指示に見える |
| **効能効果** | 「○○に効く」「○○を改善」「○○を治す」 | **最も違反しやすいポイント** |

#### 食品で言ってはいけない表現（NG一覧）

**身体の構造・機能への効果:**
- 「体質が改善する」「代謝を高める」「脂肪を燃焼する」
- 「血液をサラサラにする」「血糖値を下げる」「コレステロールを下げる」
- 「免疫力を高める」「腸内環境を改善する」「便秘を改善する」
- 「デトックス効果がある」「体内の毒素を排出する」

**疾病の治療・予防:**
- 「がんに効く」「高血圧の改善」「生活習慣病の予防」
- 「動脈硬化を防ぐ」「糖尿病に良い」「アレルギーが治る」
- 「更年期障害に効く」「痛風の予防」

**身体の特定部位への作用:**
- 「膝に効く」「腸を整える」「肝臓を守る」
- 「お腹の脂肪を落とす」「二の腕が細くなる」

**ダイエット系:**
- 「痩せる」「体重が減る」「○kg減量」「食欲を抑える」
- 「脂肪燃焼効果、絶大！」「飲むだけで痩せる」
- 「○○の働きで体内の余分な脂肪を分解し、体外に排出する」

#### 食品で言ってよい表現（OK一覧）

**栄養補給・健康維持:**
- 「不足しがちな栄養を補う」「栄養バランスを整える」
- 「毎日の健康のためにお召し上がりください」
- 「中高年からの栄養補給に」
- 「多忙で食事が不規則な方の栄養補給に」

**成分の事実記述:**
- 「○○に含まれるグルコサミンは、身体の重要な構成成分です」
- 「1粒にレモン10個分のビタミンC」
- 「食物繊維○g配合」（事実に基づく場合）

**抽象的なサポート表現:**
- 「ダイエットサポート」「健康的な生活習慣をサポート」
- 「食生活の補助に」「運動や食事と合わせて、体作りをサポート」
- 「カロリーコントロールの強い味方」

**気分・生活の表現（身体の変化を伴わないもの）:**
- 「朝からスッキリ」「毎日の元気に」「リフレッシュ」
- 「燃えるあなたに」「パワーチャージ」「エネルギー補給」

**用法の表現:**
- OK: 「目安として1〜2粒」「1日10粒を上限に」
- NG: 「食事の前に」「寝る前に」「毎日3粒」（医薬品的な指示）

#### 置き換えダイエット食品の特例

低カロリー食品への置き換えにより、**摂取する総カロリーが減少して結果的に痩せること**は医薬品的な効能効果ではない。置き換えダイエット食品に限り「ダイエット」「痩せる」の表現が一定程度可能。ただし、身体の特定部位や機能への影響は言えない。

#### 違反時のペナルティ

- **刑事罰**: 2年以下の懲役もしくは200万円以下の罰金
- **課徴金**: 違反で得た売上の4.5%（最長3年遡及）
- **「何人規制」**: アフィリエイターも処罰の対象

#### 実際の摘発事例

| 時期 | 事例 | ポイント |
|------|------|---------|
| 2020年7月 | 健康食品「肝パワーEプラス」のアフィリエイト広告で広告主・広告代理店・制作会社社員が同時逮捕 | 記事態広告でも摘発される |
| 2021年3月 | 個人アフィリエイターが「更年期障害・糖尿病・痛風の予防・改善に効く」とサプリを紹介し書類送検 | **商品が3個しか売れていなくても摘発** |
| 2023年2月 | 未承認医薬品のアフィリエイト広告で逮捕 | 個人でも逮捕事例あり |

> 参考: [A8.net 薬機法にまつわる表現ルール 健康食品編](https://www.a8.net/compliance/pmd-rules-healthyfood.php)、[レントラックス 健康食品広告のルール](https://www.rentracks.co.jp/affiliate/column/00036.html)

### 9.3 景品表示法 — 食品・ダイエット関連の優良誤認・有利誤認

#### 概要

食品・ダイエット関連商品について、実際の品質・効果を大きく上回る表示をした場合、**優良誤認表示**として措置命令の対象となる。アフィリエイターが作成した広告も広告主の表示とみなされる。

#### 食品・ダイエット関連の違反事例

| 事例 | 違反内容 | 処分 |
|------|---------|------|
| 糖質カット炊飯器（2024年） | 「糖質5割カット」と表示したが性能の裏付けなし | 4事業者に措置命令 |
| ダイエットプレミアム「炭クレンズ」 | アフィリエイトサイトで「食事制限・運動なしで短期間で痩身効果」と表示 | 合理的根拠なしで優良誤認 |

#### 不実証広告規制（第7条第2項）

消費者庁は事業者に対し、表示の裏付けとなる合理的な根拠を示す資料の提出を求めることができる。**15日以内に資料を提出できなければ不当表示とみなされる**。

#### NG表現パターン

- 「食事制限なしで○kg痩せる」（合理的根拠が必要）
- 「飲むだけで脂肪が落ちる」（実証不可能）
- 「日本一の品質」「業界No.1」（客観的根拠なし）
- 「医師推薦」「専門家おすすめ」（根拠・文脈なし）
- 報酬額順のランキングを「人気ランキング」と偽る

> 参考: [消費者庁 景品表示法](https://www.caa.go.jp/policies/policy/representation/fair_labeling)

---

### 9.4 食品表示法 — アフィリエイトブログへの適用

#### 概要

食品表示法は主に**食品の容器包装に表示される情報**を規制する法律であり、アフィリエイトブログの記事内容を直接規制するものではない。ただし、以下の点でブログ記事に間接的に関係する。

#### アフィリエイトブログに関係する部分

| 項目 | 内容 |
|------|------|
| 栄養成分表示 | 記事内で栄養成分の数値を記載する場合、商品パッケージの表示と一致させること |
| アレルゲン表示 | グルテンフリー食品等でアレルゲンの有無に言及する場合、誤情報は消費者に健康被害を与える可能性 |
| 機能性表示食品の表示 | 消費者庁に届出済みの表示内容を超えた表現はNG |
| 特別用途食品 | 「アレルゲン除去食品」「病者用食品」等の名称を許可なく使用してはならない |

#### 注意事項

- 食品表示法は容器包装の規制が中心だが、ブログで**商品の栄養成分やアレルギー情報を誤って記載**すると、消費者に実害を与え、他の法律（景品表示法・健康増進法）で問題になる
- 機能性表示食品の届出内容をブログで紹介する場合は、**届出表示そのままの文言を引用**し、ブログ側で拡大解釈しないこと

---

### 9.5 特定商取引法 — 健康食品アフィリエイトでの注意点

#### 概要

特定商取引法（特商法）は通信販売等の取引を規制する。アフィリエイターは「広告を行う者」として間接的に関わる。

#### アフィリエイトブログに関係する主な規制

| 規制 | 内容 | アフィリエイターへの影響 |
|------|------|----------------------|
| 誇大広告の禁止（第12条） | 著しく事実に相違する表示、著しく優良・有利と誤認させる表示の禁止 | ブログの記事内容が対象 |
| 定期購入の表示義務 | 定期購入が条件の場合、その旨・総額・解約条件を明記 | ブログで定期購入商品を紹介する際は条件を正確に記載 |
| 最終確認画面の表示（第12条の6） | 2022年改正で追加。申込の最終確認画面での表示義務 | 直接的にはECサイト側の義務だが、ブログで誤った条件を記載するとトラブルに |

#### 2024〜2025年の動向

- 2024年以降、健康食品の定期購入に関する行政処分が集中
- 特に「初回お試し○円」で誘引し、実際は定期購入が条件というケースが多数処分
- アフィリエイト記事で「初回○円」と強調し、定期購入の条件を小さく書く/書かないのは**特商法違反のほう助**となるリスクがある

#### ブログ記事での対応

- 定期購入商品を紹介する場合は、**初回価格だけでなく2回目以降の価格・総額・最低継続回数・解約方法**を明記する
- 「いつでも解約可能」と書く場合は、実際の解約条件（電話のみ・営業時間内のみ等）も正確に記載する

### 9.6 機能性表示食品・特定保健用食品（トクホ）の制度と記事での扱い

#### 3つの制度の違い

| 項目 | 特定保健用食品（トクホ） | 機能性表示食品 | 一般食品（健康食品含む） |
|------|----------------------|--------------|----------------------|
| 審査 | **国が個別審査・許可** | 企業が届出（国は審査しない） | なし |
| 根拠 | 臨床試験が必須 | 臨床試験 or 研究レビュー | 不要（ただし根拠なく効果を謳えない） |
| 表示できる効果 | 許可された保健の用途 | 届出された機能性 | **効果効能の表示は一切不可** |
| マーク | トクホマーク（許可証票） | なし | なし |
| 効果の例 | 「おなかの調子を整える」「コレステロールの吸収を抑える」 | 「本品にはGABAが含まれます。GABAには睡眠の質の向上に役立つ機能があることが報告されています」 | 言えない |

#### アフィリエイト記事で守るべきルール

**トクホ商品を紹介する場合:**
- 許可された保健の用途の範囲内でのみ効果を記載
- 許可されていない効果を追加で謳ってはならない
- 「国が許可した」ことは事実として記載可
- 「トクホだから安心・安全」等の誇大表現はNG

**機能性表示食品を紹介する場合:**
- 届出された機能性表示の範囲内でのみ効果を記載
- **「国が評価したものではない」旨の注記が必要**（2024年9月施行の制度改正で表示方法が見直された）
- 「疾病の診断、治療、予防を目的としたものではない」旨の注記も必要
- 届出内容を超えた効果の拡大解釈はNG

**一般食品（健康食品・サプリメント）を紹介する場合:**
- **効果効能の表示は一切不可**
- 栄養成分の含有情報、使用感、味・食感の感想に留める

#### 2024年の紅麹問題の影響

2024年に発生した紅麹サプリメント問題を受け、機能性表示食品制度が見直された。事業者の安全性確保の義務が強化されており、記事で「機能性表示食品だから安全」と断定することはさらにリスクが高まっている。

> 参考: [消費者庁 機能性表示食品の今後について（PDF）](https://www.caa.go.jp/policies/policy/food_labeling/foods_with_function_claims/movie_004/assets/food_labeling_cms206_20250106_04.pdf)

### 9.7 ASPルール（健康食品・ダイエット系案件の特有ルール）

#### A8.net — 健康食品カテゴリの追加ルール

A8.netは薬機法・景品表示法・健康増進法に基づく独自のコンプライアンスガイドを公開している。

**ジャンル別NG/OK表現（健康食品）:**

| ジャンル | NG表現 | OK表現 |
|---------|--------|--------|
| ダイエット | 「痩せる」「細くなる」「○kg減」「脂肪燃焼」「代謝促進」「デトックス」「メタボ」 | 「栄養補給」「健康維持」「ダイエットサポート」「燃えるあなたに」 |
| 腸内環境 | 「腸活」「便秘」「善玉菌が増える」「悪玉菌が減る」「宿便」 | 「朝からスッキリ」「善玉菌を補う」「爽快気分」 |
| 免疫・血行 | 「免疫力が上がる」「血圧が高めな方に」「血液サラサラ」「血糖値が気になる方に」 | 「毎日の元気に」「栄養補給」「年齢とともに不足しがちな成分」 |
| 筋肉 | 「筋力アップ」「脂肪燃焼」「疲労/ダメージ」「修復/回復」 | 「パワーチャージ」「鍛える男に」「エネルギー補給」 |
| 脳・神経 | 「睡眠サポート」「イライラが落ち着く」「精神安定」「自律神経」 | 「毎日の元気に」「気分スッキリ」「リフレッシュ」 |
| 体臭・口臭 | 「消臭」「口臭」「体臭」「腸内環境改善」 | 「○○の香りで息リフレッシュ」「エチケット対策」 |

**体験談のルール:**
- 体験者の存在やコメントの捏造は**絶対NG**
- サプリ摂取だけで痩せた、肌や髪に効果があったという体験談は**NG**
- 都合の良い体験談のみの引用は**NG**
- メリットだけ断定的でデメリット情報がない記事は**NG**
- 「※あくまで個人の感想であり、効果効能を保証するものではありません」の打ち消し表示をつけても**免責にならない**

**ビジュアル表現:**
- 健康食品のBefore/After画像は**原則使用禁止**
- 「○kg痩せた」等の効果を示唆する口コミは「個人感想」注釈付きでも**使用不可**

#### もしもアフィリエイト — 健康食品関連の注意点

もしもアフィリエイトには健康食品に特化した独自ガイドラインの公開は確認されていないが、**利用規約において薬機法・景品表示法・健康増進法の遵守を求めている**。基本的なNG/OK表現はA8.netと同様の基準で運用すべき。

> 参考: [A8.net 法律関連の禁止事項](https://www.a8.net/compliance/legal-prohibitions.php)、[A8キャンパス 健康食品アフィリエイトの法律と注意すべき表現](https://www.a8.net/campus/campus-blog/3662-healthy-food.html)

### 9.8 食事・健康法カテゴリ — テーマ別NG/OK表現

#### グルテンフリー食品の効果に関する表現

| 書きたいこと | NG表現 | OK表現 |
|-------------|--------|--------|
| 健康効果 | 「グルテンフリーにすると体調が良くなる」「腸の炎症が治まる」 | 「グルテンを含まない食品です」（事実記述のみ） |
| アレルギー対応 | 「小麦アレルギー対応食品」（特別用途食品の許可なし） | 「小麦を使用していない食品です」「原材料に小麦を使用していません」 |
| ダイエット効果 | 「グルテンフリーで痩せる」「小麦をやめたら痩せた」 | 「小麦を使わない食事スタイルです」 |
| 体質改善 | 「体質が変わった」「肌荒れが治った」 | 「食の選択肢の一つとしてご紹介します」 |

**重要な注意:**
- 日本には「グルテンフリー」の法的な定義・基準がない（海外では20ppm以下等の基準がある）
- グルテンフリー表示は「小麦アレルゲンを含まないことを示すものではない」
- 小麦アレルゲンを含む食品に「グルテンフリー」と表示することは景品表示法上問題
- 医学的にグルテンに問題のない健常者がグルテンフリー食を実践しても**科学的に確立された健康効果はない**（2019年 Digestive Diseases and Sciences誌レビュー）
- セリアック病・グルテン過敏症の患者への医学的効果を一般化してはならない

#### 糖質制限に関する表現

| 書きたいこと | NG表現 | OK表現 |
|-------------|--------|--------|
| ダイエット効果 | 「糖質制限で確実に痩せる」「糖質オフで○kg減」 | 「糖質量を意識した食事スタイルです」 |
| 血糖値 | 「血糖値を下げる」「血糖値コントロール」 | 「糖質量が少ない食品です」（成分の事実記述） |
| 健康効果 | 「糖質制限で病気が予防できる」「生活習慣病に効く」 | 「糖質の摂取量について考える機会に」 |
| 体験談 | 「糖質制限で10kg痩せました」 | 「糖質を意識した食事を心がけています」（身体の変化には触れない） |

#### 特定の食べ物・飲み物の「健康効果」に関する表現

| 書きたいこと | NG表現 | OK表現 |
|-------------|--------|--------|
| 食品の効能 | 「トマトのリコピンが血管を守る」「ブルーベリーが目に効く」 | 「トマトにはリコピンが含まれています」（成分の事実記述） |
| 飲み物の効能 | 「緑茶を飲むとがんになりにくい」「ルイボスティーで老化防止」 | 「緑茶にはカテキンが含まれています」（成分の事実記述） |
| スーパーフード | 「○○は奇跡の食材」「万能食品」 | 「○○は△△という栄養素を含む食品です」 |
| 食品の比較 | 「○○は他の食品より健康効果が高い」 | 「○○に含まれる△△は100gあたり○mgです」（数値比較は事実に基づけばOK） |

#### ダイエット効果に関する表現

| 書きたいこと | NG表現 | OK表現 |
|-------------|--------|--------|
| 痩身効果 | 「飲むだけで痩せる」「食べて痩せる」 | 「食生活の見直しをサポート」 |
| 体重減少 | 「○週間で○kg減」「ウエスト○cm減」 | 「カロリーコントロールの参考に」 |
| 部分痩せ | 「お腹の脂肪を落とす」「二の腕が細くなる」 | 使用不可（食品では部分痩せの表現は一切NG） |
| メカニズム | 「脂肪を分解して体外に排出」「代謝を促進して脂肪を燃焼」 | 「低カロリーな食事の選択肢として」 |

#### 「○○に効く」「○○が治る」の表現

**食品について「効く」「治る」は一切使えない。** これは薬機法における最も基本的な禁止事項。

| NG表現 | 違反する法律 |
|--------|------------|
| 「○○が△△に効く」 | 薬機法（未承認医薬品の広告） |
| 「○○で△△が治る」 | 薬機法（未承認医薬品の広告） |
| 「○○が△△を予防する」 | 薬機法 + 健康増進法 |
| 「○○で△△が改善する」 | 薬機法 + 健康増進法 |
| 「○○は△△にいい」 | 文脈により薬機法違反の可能性 |

**言い換えの方向性:**
- 「○○には△△という成分が含まれています」（成分の事実記述に留める）
- 「○○は栄養豊富な食品です」（一般的な栄養表現に留める）
- 「○○をお試しください」（効果には触れない）

#### 体験談の書き方（食事・健康法の場合）

**絶対にNGな体験談:**
- 「○○を食べ始めてから、体重が減りました」
- 「△△を飲んだら、血圧が下がりました」
- 「糖質制限を始めたら肌荒れが治りました」
- 「グルテンフリーにしたら体調が良くなりました」
- 「このサプリを飲み始めてから、痩せ始めた」

**OKな体験談:**
- 「○○は味が良くて続けやすいです」（味・食感の感想）
- 「△△は料理に使いやすい食材です」（利便性の感想）
- 「このレシピは手軽に作れました」（調理の感想）
- 「糖質を意識した食事を心がけています」（生活スタイルの紹介、効果には触れない）
- 「グルテンフリーの食品が増えて選択肢が広がりました」（市場・製品の紹介）

**注意:**
- 「※あくまで個人の感想であり、効果効能を保証するものではありません」の打ち消し表示は**免罪符にならない**
- 効果・性能が全く得られない人が相当数いるにもかかわらず、効果があったとする体験談を掲載した場合、打ち消し表示があっても**景品表示法上問題になる**

### 9.9 ステマ規制との関係（再掲・食事健康法カテゴリ固有の注意点）

食事・健康法カテゴリにおいても、セクション2で記載したステマ規制がすべて適用される。

**追加注意点:**
- 食品メーカーから商品提供を受けてレビュー記事を書く場合、PR表記は必須
- 「友人に勧められた」等の形式で広告であることを隠すのはステマ規制違反
- アフィリエイトリンクを含む食品レビュー・レシピ記事はすべて「広告」表記が必要

### 9.10 食事・健康法カテゴリ — 記事執筆チェックリスト

記事を公開する前に、以下のすべてを確認すること。

- [ ] 食品について「効く」「治る」「改善する」「予防する」の表現を使っていないか
- [ ] 身体の構造・機能への直接的な効果を謳っていないか
- [ ] 疾病名（糖尿病、高血圧、がん等）を効果と結びつけていないか
- [ ] ダイエット効果について「痩せる」「体重が減る」等の断定的表現を使っていないか
- [ ] 体験談が「効果効能」ではなく「味・食感・使い勝手」に限定されているか
- [ ] Before/After写真を使用していないか
- [ ] 「個人の感想です」の打ち消し表示で効果効能の表現を許容していないか
- [ ] 機能性表示食品・トクホの場合、届出/許可された範囲の表現に留まっているか
- [ ] 機能性表示食品の場合、「国が評価したものではない」旨の注記があるか
- [ ] 定期購入商品の場合、条件（総額・最低回数・解約方法）を正確に記載しているか
- [ ] ランキングの根拠が明確で、報酬額順になっていないか
- [ ] グルテンフリーの健康効果を断定していないか
- [ ] PR表記が記事のファーストビューに配置されているか
- [ ] 引用している栄養成分の数値が正確か
- [ ] 「最高」「日本一」「業界No.1」等の最上級表現に客観的根拠があるか
